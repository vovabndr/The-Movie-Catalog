# The Movie Catalog

"The movie catalog" is created  to search movies, view the information about  a movie title, poster, release date, description, rating). Search includes three modes such as search by name, by year and by genre. Also you can create your "favorites list". To do this, just click "Add to Favorite" when you see the information about the movie. The movie will immediately appear in the "Favorites" tab. You can delete it from this list at any time. To do this, use the "Remove from Favorite" button. If you don't know what movie to watch, look for something interesting in the "Top rated" tab, where you will see the most popular cinema novelties.
